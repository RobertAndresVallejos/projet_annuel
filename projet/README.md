# Projet annuel

## Description du projet
Ce projet est une application web d'e-commerce pour la vente de vêtements ("ThatWay"), développée en **PHP** et **SQL**. Il implémente un système complet de gestion utilisateur sécurisée, un catalogue de produits avec filtrage, un panier dynamique (gestion de session), et des fonctionnalités de communication client (contact et espace personnel).

## Langages utilisés
- HTML 
- CSS
- PHP
- SQL
- JS

## Outils utilisés
- MAMP
- MySQL
- PHPMailer
